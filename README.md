# Class Incremental Learning for Light-weighted Networks
Implementation of the paper "Class Incremental Learning for Light-weighted Networks".

## Requirements
- Python 3.9
- PyTorch 1.13.1
- CUDA 11.6

## Datasets
> #### 1. CIFAR100
> #### 2. Tiny-ImageNet
> #### 3. ImageNet-subset
